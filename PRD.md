# 产品需求文档 (PRD)：Luffa RWA Insight 小程序

## 1. 文档概述

- **产品名称**：RWA Insight (Luffa Mini-Program)
    
- **版本**：V1.0.0
    
- **状态**：初稿
    
- **目标**：为 Luffa 用户提供一站式的现实世界资产（RWA）链上数据监控、分析及投资入口。
    

---

## 2. 产品定位

利用 Luffa 的多链钱包能力和社交属性，将 RWA.xyz 的复杂数据转化为移动端友好的交互界面，帮助用户快速捕捉私人信贷、代币化国债及房地产等赛道的投资机会。

## 4. 技术需求 (Luffa SDK 集成)

| **功能模块** | **技术实现要点**                                                     |
| -------- | -------------------------------------------------------------- |
| **容器适配** | 使用 H5 响应式布局，适配 Luffa `superbox-mini-sdk` 环境。                   |
| **钱包连接** | 调用 `luffa.getAccount()` 获取当前登录地址。                              |
| **链上交互** | 调用 `luffa.sendTransaction()` 支持在小程序内直接参与 RWA 质押。               |
| **分享功能** | 集成 `luffa.setShareInfo()`，支持用户将特定数据看板分享至 Luffa 群聊。             |
| **数据源**  | 初始阶段通过 `fetch` 接入 RWA.xyz 或 Dune Analytics API；二期考虑接入 Indexer。 |
